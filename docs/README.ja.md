# Image-Tools

最先端のAI技術を搭載した、包括的かつプロダクショングレードの画像処理ツールキット。プロフェッショナルな背景除去、ウォーターマーク除去、スマート画像分割を提供し、完全にローカルで実行されプライバシーを保護します。

## ✨ 主な機能

### 🎯 背景除去
- **Ultraバックエンド** - BRIA RMBG-2.0による最高品質（非商用）
- **Unifiedバックエンド** - BiRefNet-massiveによる商用対応（MITライセンス）
- **高度な処理** - Trimap精細化、マルチカラースペースdefringing
- **カラー最適化** - 緑/白/黒の背景に特化した処理
- **制御可能な強度** - 除去の強さを細かく調整（0.1-1.0）

### 🌊 ウォーターマーク除去
- **Geminiウォーターマークリムーバー** - Google Gemini AIが生成したウォーターマークを除去
- **逆アルファブレンディング** - 数学的に正確な除去
- **自動検出** - ウォーターマークサイズの自動検出（48px/96px）
- **調整可能な強度** - 部分的または完全な除去

### ✂️ 画像分割
- **スマートグリッドレイアウト** - 最適なグリッド構成の自動計算
- **カスタム配置** - 柔軟な行×列の構成
- **品質保持** - 非圧縮、ロスレス分割
- **バッチ処理** - 複数の画像を一度に分割

## 🚀 クイックスタート

### インストール

要件：Python 3.13+ と `uv` パッケージマネージャー

```bash
git clone https://github.com/yourusername/Image-Tools.git
cd Image-Tools
uv sync
```

### 実行

```bash
uv run main.py
```

対話的なプロンプトに従ってください：
1. **フォルダを選択** - 入力ディレクトリを選択
2. **操作を選択** - 背景除去、ウォーターマーク除去、画像分割
3. **設定を構成** - ユースケースに応じてパラメータを調整
4. **処理** - ツールに残りを任せる

出力ファイルは `<入力フォルダ>/output/` に保存されます

## 🔧 技術詳細

### パフォーマンス

- **単一画像**：2-5秒（解像度とハードウェアに依存）
- **メモリ使用量**：~3-4GB GPU（BiRefNet-massive）、~2GB CPU
- **バッチ処理**：効率的なセッション再利用
- **品質**：商用サービスと同等以上

### ハードウェア要件

| 構成 | スペック |
|---------------|-------|
| **最小** | 8GB RAM + CPU（低速） |
| **推奨** | 16GB RAM + 4GB+ VRAM GPU（CUDA） |
| **最適** | 32GB RAM + 8GB+ VRAM GPU |
