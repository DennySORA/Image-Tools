    feat(backend): add unified SOTA background removal solution
    
        Implement comprehensive unified backend based on latest research:
    
        Core Features:
        - BiRefNet-massive: Primary segmentation model (MIT license)
        - Auto Alpha Matting: Intelligent edge refinement (strength ≥ 0.3)
        - Edge Defringing: Remove color spill/contamination
        - Optional Color Filter: Pure black/white/green background optimization
    
        Technical Implementation:
        - Multi-stage pipeline: segmentation → color filter → defringing
        - Controllable strength parameter (0.1-1.0)
        - Dynamic matting threshold calculation
        - HSV/LAB color space detection for accurate filtering
        - Morphological operations for mask refinement
    
        UI/UX Changes:
        - Simplified background removal to single unified option
        - Added color filter configuration (off by default)
        - Background color selection: green/white/black
        - Updated ProcessConfig to support extra_config
    
        Benefits:
        - Better edge quality (hair, fur, transparent materials)
        - No more white/green edge fringing
        - Optimized for pure-color backgrounds
        - Fully local, no API calls needed
        - MIT license - commercially usable
    
        Documentation:
        - Comprehensive UNIFIED_BACKEND.md guide
        - Usage examples and FAQ
        - Performance benchmarks
        - Troubleshooting guide

        - BiRefNet: https://github.com/ZhengPeng7/BiRefNet
        - Rembg: https://github.com/danielgatis/rembg
